# dincertekin.com
Personal portfolio website built with Svelte and SvelteKit.

## Description
dincertekin.com is my personal portfolio website that showcases my GitHub projects, Medium posts, and a general introduction about me. Built with Svelte 5 and SvelteKit for the frontend, styled with Tailwind CSS, and hosted on Cloudflare.

## Getting Started

### Dependencies
* Node.js and npm or yarn
* Cloudflare account (for deployment)

### Installing

* Clone the repository:
```bash
git clone https://github.com/dincertekin/dincertekin.com.git
cd dincertekin.com/
```

* Install dependencies:
```bash
npm install
```

### Executing program

* Start the development server:
```bash
npm run dev
```

* Build for production:
```bash
npm run build
```

* Deploy to Cloudflare:
```bash
npm run deploy
```

## Help
If you run into issues during deployment, make sure you are logged in to Cloudflare via Wrangler.

```bash
npx wrangler login
```
